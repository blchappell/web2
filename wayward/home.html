Issue #4: The Fear of Failure

“When did we start using failure as a stopping point, particularly in our creative process?”

"If at first you don’t succeed, try again.” We’ve all grown up hearing this statement, as a way to regain optimism after not meeting our set expectations and feeling like we’ve failed at what we worked for. But like many things, this is easier said than done. While this primitive idea seems like common sense,

sometimes it’s incredibly hard for us to accept when we’ve spent countless efforts on one thing to have it go no where. So this calls into question: when did we start using failure as a stopping point, particularly in our creative process?
